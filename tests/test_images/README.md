# Test Images Directory
# ======================

# 2. Run inference: python src/inference.py --image tests/test_images/your_image.jpg
# 3. Or run the API and test with these images

# - pothole_example.jpg: Example of a clear pothole
# - normal_road.jpg: Example of a road without potholes
# - multiple_potholes.jpg: Image with multiple potholes
